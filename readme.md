- **cat-api**

  api模块，推荐post-body-json的方式传入数据，支持get?params=URIEncoder.encode({参数})。

  接口路径：/interfaceAction	//对于所有的api路径是一样的

  例子参数: {interId:"100001",version:1,params:{size:1,page:15}} //其中100001是接口编号，不用的api对应不用的业务类，如果想在
